# LIFTOFF X assets

Drop the artwork + audio here with these exact filenames (see
`src/themes/liftoffx/theme.ts`). Until then, the `liftoffx` theme is a stub.

## Images (.webp)
- `logo.webp`            — game logo (transparent)
- `landing.webp`         — full landing page (phone aspect, text baked in)
- `launchpad.webp`       — idle/running scene (rocket on the pad)
- `explosion.webp`       — crash scene (rocket blows up) — landscape ~1.35:1
- `eject.webp`           — win scene (clean eject) — landscape ~1.35:1
- `icons/home.webp`, `icons/history.webp`, `icons/rocket.webp`, `icons/leaderboard.webp`, `icons/chat.webp`

## Audio (.mp3)
- `audio/motif-low.mp3`, `audio/motif-high.mp3`  — tension bed (loops)
- `audio/lobby.mp3`        — ambient between rounds (loops)
- `audio/eject.mp3`        — cash-out sfx
- `audio/explosion.mp3`    — crash sfx
- `audio/countdown.mp3`    — betting countdown tick (~4.6s)
- `audio/voice-win-1..5.mp3`, `audio/voice-crash-1..5.mp3` — random voice lines
